# Kawthar Alkhawajah Portfolio (Assignment 1)

## Project Description
A simple, responsive personal portfolio website built as the starting point of my professional portfolio.  
The site includes three main sections: **About Me**, **Projects**, and **Contact**, with basic interactivity using JavaScript.

## Features
- Responsive layout (works on desktop, tablet, and mobile)
- Clean UI with CSS (Flexbox/Grid)
- Dark/Light theme toggle
- Interactive greeting / UI effects (JavaScript)
- Contact form (front-end only, no backend)

## Setup Instructions (Run Locally)
1. Download or clone the repository:
   - Click **Code → Download ZIP** and extract it, OR:
   - `git clone (your-repo-link)`
2. Open the project folder.
3. Run the website:
   - Option A: Open `index.html` directly in your browser.
   - Option B (recommended): Use **VS Code Live Server**  
     - Install the Live Server extension  
     - Right-click `index.html` → **Open with Live Server**
